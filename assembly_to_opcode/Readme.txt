################################################################
Este script se encarga de convertir algunas de las instrucciones
del procesador MIPS de 32 bits.
################################################################
COMO USAR:

se recomienda contar con un editor de texto, con el cual
escribir las intrucciones. Se incluye un ejemplo de
muestra de como funciona la herramienta, es necesario dejar una
salto de linea al final del archivo de texto (tal como el
ejemplo) para su correcto funcionamiento. Al momento de ingresar
el nombre del archivo debe incluirse la extension del mismo.
################################################################
Hecho en python 3.9.5
################################################################
